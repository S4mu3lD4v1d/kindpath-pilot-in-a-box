# KindPath — Pilot in a Box

A practical package for running a KindPath pilot:
- Setup steps
- Governance patterns
- Measurement approach
- Reporting templates

## Who this is for
Councils, schools, community groups, NGOs, and small programs that want a structured pathway from intention to measurable outcomes.

## How to use
Start at `/01-overview` and follow the sequence.

## Repository structure (recommended)
- /01-overview
- /02-setup
- /03-measurement
- /04-governance
- /05-reporting
- /templates
